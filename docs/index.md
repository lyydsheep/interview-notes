## 📖 食用指南

1. **选择一个知识点** - 点击上方卡片进入对应技术分类
2. **阅读知识点** - 浏览面试题和答案解析

4. **拓展学习** - 深入理解底层原理

## 📝 题库目录

### MySQL 数据库（9大主题）

1. **SQL 基础与范式** - 数据库设计原理
2. **存储引擎与数据类型** - InnoDB vs MyISAM
3. **索引（重点）** - B+ 树、覆盖索引、最左匹配
4. **事务（重中之重）** - ACID、MVCC、隔离级别
5. **锁机制** - 行锁、表锁、间隙锁、死锁
6. **日志系统** - Redo、Undo、Binlog
7. **性能优化** - 慢查询、索引优化、执行计划
8. **高可用与扩展** - 主从复制、读写分离、分库分表
9. **综合与思考题** - 实战案例分析
