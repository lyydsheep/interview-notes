# 面试八股文 📚

技术面试题库，包含 MySQL、Redis、操作系统等技术的面试题。

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 本地开发
```bash
npm run docs:dev
```

### 构建静态文件
```bash
npm run docs:build
```

### 预览构建结果
```bash
npm run docs:serve
```

## 📖 技术栈

- VitePress - 静态文档生成器
- Vue 3 - 前端框架

## 📁 项目结构

```
interview-notes/
├── docs/                      # 文档目录
│   ├── .vitepress/           # VitePress 配置
│   ├── index.md              # 首页
│   ├── mysql/                # MySQL 题库
│   ├── redis/                # Redis 题库
│   └── os/                   # 操作系统题库
├── vercel.json               # Vercel 部署配置
└── package.json              # 项目配置
```
